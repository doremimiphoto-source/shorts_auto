# 유튜브 쇼츠 폰트·사이즈·배치·모션 최종 제작 가이드

> 목적: 쇼츠 영상 제작 시 콘텐츠 유형별로 **어떤 폰트를 사용하고, 몇 px 크기로 넣고, 어디에 배치하며, 어떤 움직임 효과를 적용할지** 바로 참고하기 위한 최종 제작용 가이드입니다.  
> 기준 해상도: **1080 × 1920px / 9:16 세로 영상**  
> 추천 콘텐츠: 명언, 감성 문구, 위로 메시지, 동기부여, 정보성 쇼츠, 스토리텔링 쇼츠  
> 폰트 기준: **오픈소스 또는 무료 상업 사용 가능 폰트 중심**  
> 제작 도구: CapCut, Canva, Premiere Pro, After Effects, VN, Vrew 등

---

## 1. 최종 결론

쇼츠 자막은 **크게 보이는 것**보다 **모바일에서 빠르게 읽히면서 영상 분위기를 해치지 않는 것**이 중요합니다.

기존 기준은 가독성을 위해 다소 크게 잡은 편이므로, 실제 제작에서는 아래 기준을 최종값으로 사용하는 것을 권장합니다.

```text
후킹 문장: 88~96px
본문 자막: 68~72px
강조 문장: 80~88px
강조 단어: 104~112px
CTA 문구: 48~54px
출처·작가명: 36~40px
썸네일 문구: 120~140px
```

가장 추천하는 기본 조합은 다음과 같습니다.

```text
후킹:
Pretendard ExtraBold / 92px / 흰색 / 검정 외곽선 6px

본문:
Pretendard Bold / 70px / 흰색 / 검정 외곽선 5px

강조:
Pretendard Black / 108px / 노랑 또는 연분홍 / 검정 외곽선 6px

마무리:
Gowun Dodum / 66px / 아이보리 또는 흰색 / 약한 그림자

작가명·출처:
Noto Serif KR / 38px / 흰색 또는 아이보리
```

## 3. 추천 오픈소스 기반 폰트

### 3-1. Pretendard

쇼츠 메인 자막으로 가장 추천합니다.  
한글, 숫자, 영문이 안정적으로 보이며 감성형, 정보형, 동기부여형 모두에 잘 맞습니다.

| 사용 위치 | 추천 굵기 | 최종 추천 크기 |
|---|---:|---:|
| 첫 후킹 문장 | ExtraBold | 88~96px |
| 강한 후킹 문장 | Black | 96~108px |
| 일반 본문 | Bold | 68~72px |
| 강조 문장 | ExtraBold | 80~88px |
| 강조 단어 | Black | 104~112px |
| CTA 문구 | Bold | 48~54px |
| 썸네일 문구 | Black | 120~140px |

추천 용도:

- 감성 명언
- 동기부여
- 자기계발
- 정보성 쇼츠
- 썸네일 문구
- 채널 공통 자막 스타일

---

### 3-2. Noto Sans KR

정보성, 설명형, 튜토리얼형 쇼츠에 적합합니다.  
정리된 느낌을 주기 좋고, 긴 설명 자막에서도 안정적입니다.

| 사용 위치 | 추천 굵기 | 최종 추천 크기 |
|---|---:|---:|
| 제목 | Bold / Black | 86~92px |
| 본문 설명 | Medium / Bold | 58~66px |
| 단계 번호 | Bold | 76~86px |
| 보조 설명 | Medium | 40~46px |

추천 용도:

- 정보성 쇼츠
- 꿀팁 정리
- 순서 설명
- 튜토리얼
- 비교 콘텐츠

---

### 3-3. Gowun Dodum

부드럽고 따뜻한 감성을 줄 수 있는 폰트입니다.  
굵기가 강하지 않으므로 후킹보다는 본문 일부, 마지막 여운 문장, 힐링 문구에 적합합니다.

| 사용 위치 | 추천 굵기 | 최종 추천 크기 |
|---|---:|---:|
| 감성 본문 | Regular | 68~72px |
| 마지막 여운 문장 | Regular | 64~68px |
| 짧은 한 줄 명언 | Regular | 72~78px |
| 작가명 | Regular | 36~40px |

추천 용도:

- 위로 문구
- 힐링 명언
- 봄·노을·바다·산책 영상
- 잔잔한 감성 쇼츠

---

### 3-4. Nanum Gothic

익숙하고 편안한 느낌의 폰트입니다.  
최신 감각은 Pretendard보다 약할 수 있으나, 차분한 설명형 콘텐츠에는 사용할 수 있습니다.

| 사용 위치 | 추천 굵기 | 최종 추천 크기 |
|---|---:|---:|
| 제목 | Bold | 82~92px |
| 본문 | Bold | 66~72px |
| 보조 설명 | Regular / Bold | 40~46px |

---

### 3-5. Noto Serif KR

문학적이고 진중한 분위기를 줄 수 있습니다.  
다만 모바일 화면에서는 획이 얇게 보일 수 있으므로 전체 자막보다는 인용문, 작가명, 마지막 문장에만 사용하는 것을 추천합니다.

| 사용 위치 | 추천 굵기 | 최종 추천 크기 |
|---|---:|---:|
| 마지막 명언 | SemiBold / Bold | 68~76px |
| 인용문 | SemiBold | 66~74px |
| 작가명 | Regular | 36~40px |

## 5. 콘텐츠별 최종 폰트·사이즈·배치·모션

### 5-1. 감성 명언 쇼츠

분위기: 따뜻함, 여운, 공감, 위로  
추천 길이: 18~30초  
최종 방향: 본문은 너무 크게 하지 않고, 후킹과 강조만 키웁니다.

| 요소 | 추천 폰트 | 굵기 | 최종 크기 | 위치 | 추천 모션 |
|---|---|---:|---:|---|---|
| 첫 후킹 문장 | Pretendard | ExtraBold | 88~94px | 중앙보다 약간 위 | Fade In + 약한 확대 |
| 본문 명언 | Pretendard | Bold | 68~72px | 중앙 | Fade In |
| 강조 문장 | Pretendard | ExtraBold | 80~86px | 중앙 | Scale Pop 약하게 |
| 강조 단어 | Pretendard | Black | 104~110px | 중앙 | Scale Pop |
| 마지막 여운 | Gowun Dodum | Regular | 64~68px | 중앙보다 약간 아래 | Slow Fade In |
| 작가명 | Noto Serif KR | Regular | 36~40px | 하단 안전영역 위 | Fade In |

#### 실제 적용 예시

```text
[0~2초]
요즘 마음이 지친 당신에게
Pretendard ExtraBold / 92px / Y 760

[3~10초]
괜찮아요.
조금 느려도 괜찮아요.
Pretendard Bold / 70px / Y 920

[11~18초]
당신은 지금도
충분히 잘하고 있습니다.
Pretendard ExtraBold / 82px / Y 920

[19~23초]
오늘도 나를 믿어주세요.
Gowun Dodum / 66px / Y 1100
```

#### 추천 모션

| 구분 | 권장값 |
|---|---|
| 등장 | Fade In 0.25~0.3초 |
| 강조 | Scale 95% → 105% → 100%, 0.25초 |
| 퇴장 | Fade Out 0.2초 |
| 분위기 | 부드럽고 느리게 |

## 8. 폰트 움직임 효과 가이드

자막 모션은 콘텐츠 분위기를 결정합니다.  
하지만 쇼츠에서는 자막이 너무 많이 움직이면 읽기 어렵습니다.  
따라서 모션은 짧고 명확하게 사용합니다.

### 8-1. 기본 모션 종류

| 모션 | 설명 | 추천 콘텐츠 | 권장 시간 |
|---|---|---|---:|
| Fade In | 서서히 나타남 | 감성, 명언, 힐링 | 0.2~0.5초 |
| Fade Out | 서서히 사라짐 | 대부분의 콘텐츠 | 0.15~0.4초 |
| Slide Up | 아래에서 위로 등장 | 정보성, 동기부여 | 0.2~0.3초 |
| Slide Down | 위에서 아래로 등장 | 제목, 정보형 | 0.2~0.3초 |
| Scale Pop | 살짝 커졌다 안정 | 강조 단어 | 0.15~0.3초 |
| Typewriter | 글자가 타이핑됨 | 감성, 스토리 | 문장당 0.5~1.2초 |
| Blur In | 흐림에서 선명하게 | 감성, 시네마틱 | 0.3~0.5초 |
| Bounce | 튀는 등장 | 강한 동기부여 | 0.15~0.25초 |
| Wipe Reveal | 좌우로 드러남 | 정보형, 리스트 | 0.25~0.4초 |

### 8-2. 콘텐츠별 추천 모션

| 콘텐츠 유형 | 추천 모션 | 피해야 할 모션 |
|---|---|---|
| 감성 명언 | Fade In, Slow Fade, Blur In | 과한 Bounce, 빠른 흔들림 |
| 위로·힐링 | Slow Fade, Blur In | 빠른 Slide, 강한 Pop |
| 동기부여 | Pop In, Slide Up, Scale Pop | 너무 느린 Fade |
| 정보성 | Slide Up, Wipe Reveal, Cut | Typewriter 과다 사용 |
| 스토리텔링 | Fade In, Typewriter, Scale Pop | 모든 문장 Pop 처리 |
| 썸네일 후킹 | Pop In, Scale Pop | 긴 Fade In |

### 8-3. 모션 속도 권장값

| 분위기 | 등장 시간 | 퇴장 시간 | 특징 |
|---|---:|---:|---|
| 감성형 | 0.3~0.5초 | 0.3초 | 부드럽고 느리게 |
| 정보형 | 0.2~0.3초 | 0.15~0.25초 | 빠르고 명확하게 |
| 동기부여형 | 0.15~0.25초 | 0.15초 | 강하고 짧게 |

## 11. 폰트 크기 조정 기준

같은 70px라도 폰트마다 체감 크기가 다릅니다.  
따라서 아래 기준으로 조정합니다.

| 상황 | 조정 방법 |
|---|---|
| 문장이 2줄 이상이면 | 본문 크기를 4~6px 줄임 |
| 배경이 복잡하면 | 외곽선을 1~2px 키움 |
| 글자가 너무 답답하면 | 줄 간격을 120~130%로 늘림 |
| 감성 분위기가 약하면 | 굵기를 한 단계 낮추고 Fade를 느리게 함 |
| 동기부여 느낌이 약하면 | 굵기와 크기를 한 단계 키움 |
| 모바일에서 작게 보이면 | 전체 자막을 4px 키움 |
| 화면이 촌스러워 보이면 | 색상을 줄이고 흰색 중심으로 정리 |
| 영상 배경이 밝으면 | 외곽선 또는 반투명 배경 박스 추가 |
| 영상 배경이 어두우면 | 외곽선은 줄이고 그림자만 약하게 사용 가능 |

### 폰트별 체감 크기 보정

| 폰트 | 체감 크기 | 조정 기준 |
|---|---|---|
| Pretendard | 기준값으로 적정 | 표 기준 그대로 사용 |
| Noto Sans KR | 안정적이고 정직함 | 본문 60~66px 적합 |
| Gowun Dodum | 부드럽고 약간 작게 보임 | Pretendard보다 2~4px 크게 가능 |
| Noto Serif KR | 얇게 보일 수 있음 | 본문보다 2~6px 크게 가능 |
| Nanum Gothic | 무난하지만 평범함 | Bold 사용 권장 |
